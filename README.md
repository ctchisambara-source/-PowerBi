## 📊 Power BI

🔍 Overview
Using Power BI to analyse sales data and create interactive dashboards. The work includes cleaning and modelling datasets, building visual reports, and using charts and filters to explore sales performance and trends. The dashboards demonstrate how raw data can be transformed into clear, interactive insights to support data-driven decision making.

🚀Skills
----------------

✔️Data Extraction & Transformation (power query)

Worked with data from mutliple sources, cleaned it with usable formt e.g renaming columns, correcting
data types, and combing data through merging and appending.

✔️Data Modelling

Built efficient data model by defining relationships & organising tables for reporting. Utilised DAX analysis expression to created calculated columns, aggregated values used within visualisations.

✔️Report development

Created clear , interactive visual to present data insights in an easy format whilst using slicing,filtering ,and drill down capabilities.

✔️Dashboard Creation

Created user friendly driven dashboard to support business decision making.

### 🧩 Project Value and Impact
---------
Power Bi is a leading business intelligence too widely used across industries for data analysis and reporting. This project highlight my ability to: 

- Transform and prepare data using Power query
- Build calculation and measures using DAX
- Develop interactive reports and dashboards
- Present clear insights through effective data visualization.

These capabilites are fundameral for roles such as Data Analyst/ Techinician.
